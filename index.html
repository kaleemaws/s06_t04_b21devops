What is DevOps?

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops) to shorten the software development lifecycle and deliver high-quality software continuously. DevOps emphasizes automation, collaboration, monitoring, and continuous improvement.

  • Continuous Integration & Continuous Delivery (CI/CD)
  • Infrastructure as Code (IaC)
  • Monitoring & Logging
  • Cloud & Containerization

Training Structure

Duration: 8 Weeks
Classes: 2 per week
Class Duration: 2 Hours
Mode: Hands-on + Theory

Detailed 2-Week Training Plan (Sample)

Week 2: Version Control & CI Basics

  • Class 3:
    • Git Fundamentals

Remaining weeks would cover Docker, Kubernetes, Cloud (AWS), Terraform, Monitoring, and a Capstone Project.
